# NHS Waiting List Analysis (Jan 2026)

## Overview
This project analyses NHS referral-to-treatment (RTT) waiting times across different medical specialties.

## Objective
To identify which specialties have the longest waiting times and largest patient backlogs.

## Tools Used
- Microsoft Excel (Pivot Tables, Data Cleaning, Charts)

## Data Source
NHS England RTT waiting times dataset (January 2026)

## Key Steps
- Cleaned and simplified raw NHS data
- Selected key variables (specialty, waiting list size, average waiting time)
- Created a ranked table of specialties by waiting time
- Applied conditional formatting to highlight high waiting times
- Built a chart to visualise differences across specialties

## Key Insights
- Oral Surgery and ENT services have the longest waiting times (~17 weeks)
- Trauma & Orthopaedics has the largest patient backlog (>800,000 patients)
- Lower-volume specialties tend to have shorter waiting times

## Output
A simple Excel dashboard showing waiting times by specialty.
